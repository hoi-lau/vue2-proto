# vue-proto

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### includes

 - vuex
 - vue-router
 - axios
 - stylus
 - eslint (rules) && editorconfig && prettierrc
 - babel
 - nprogress
 - husky & lint-staged
 - webpack-bundle-analyzer
